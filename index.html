// REAL-TIME EDGE AI FOR WASTE CLASSIFICATION

A fully integrated computer vision system that identifies and physically sorts plastic bottles by resin type — PET, HDPE, LDPE, PP, and UNCLASSIFIED — in real time. Trained on edge hardware using TensorFlow and Edge Impulse, the model runs inference directly on-device with no cloud round-trip. Classification output drives mechanical actuators that physically sort each bottle, enabling fully automated waste segregation and supporting sustainable waste management objectives.

Key Contributions
  • Trained a CNN classifier using MobileNet transfer learning via TensorFlow and Edge Impulse to classify 5 plastic resin types with >90% accuracy.
  • Built a continuous camera module pipeline for image capture and on-device inference with minimal latency on a Raspberry Pi 5.
  • Designed and integrated mechanical sorting actuators driven by model output, fully automating the physical sorting process.
  • Improved plastic segregation reliability, reducing contamination in recycling streams.

AI Document

// NLP + COMPUTER VISION · ZERO-SHOT FILE CLASSIFICATION CLI

A Python CLI tool that intelligently organises files in a directory using zero-shot NLP and image scene recognition — no manual labeling or predefined rules required. It reads semantic content of documents and visual context of images to determine categories, then moves files automatically. A persistent SQLite layer tracks scan history to avoid re-processing, and a scheduler enables fully hands-free recurring folder management.

Key Contributions
  • Implemented zero-shot text classification using facebook/bart-large-mnli via HuggingFace Transformers — no training data required.
  • Integrated OpenCV-based image scene recognition for classifying photos and visual files by content type.
  • Built perceptual hashing for duplicate image detection, preventing redundant file copies.
  • Designed a SQLite persistence layer for scan history enabling efficient incremental runs.
  • Added --dry-run and --watch modes for safe previewing and automated recurring execution.
