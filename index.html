Automation · NLP · Computer Vision Personal Project · 2026

AI Document
Organizer

// ZERO-SHOT FILE CLASSIFICATION · HANDS-FREE FOLDER MANAGEMENT

A Python CLI tool that automatically classifies and organises files using zero-shot NLP and image scene recognition — no manual labeling, no predefined rules. Set it to watch a folder and it handles everything: reading document content, recognising images, detecting duplicates, and filing everything correctly.

Demo Video Coming soon — showcase in progress
What it does
  • Classifies documents by reading their content using facebook/bart-large-mnli — no training data needed.
  • Recognises image types by visual content using OpenCV scene recognition.
  • Detects and skips duplicate images using perceptual hashing.
  • Runs on a schedule with --watch mode — fully hands-free once set up.
  • SQLite history layer ensures files aren't processed twice on repeat runs.

A computer vision system that classifies plastic bottles by resin type (PET, HDPE, LDPE, PP, UNCLASSIFIED) in real time and drives mechanical actuators to physically sort them — no cloud, no internet required.
