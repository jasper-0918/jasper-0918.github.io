Thesis Project Apr 2025 – Nov 2025

Automatic Plastic Bottle Segregation System

Real-time edge AI for waste classification — La Salle University Ozamiz

A fully integrated computer vision system that identifies and physically sorts plastic bottles by resin type (PET, HDPE, LDPE, PP, and UNCLASSIFIED) in real time. Trained on edge hardware using TensorFlow and Edge Impulse, the model runs inference directly on-device, eliminating latency from cloud round-trips. Classification output drives mechanical actuators that physically sort bottles, enabling fully automated waste segregation with no human intervention. The system directly supports sustainable waste management by reducing contamination in recycling streams.

Key Contributions
  • Trained a CNN classifier (MobileNet transfer learning) using TensorFlow and Edge Impulse to classify 5 plastic resin types in real time.
  • Built a continuous camera module pipeline for image capture and on-device inference with minimal latency.
  • Designed and integrated mechanical sorting actuators driven by model output, automating the physical sorting process.
  • Achieved high classification accuracy, improving segregation reliability and reducing recyclable contamination.

Personal Project 2026

AI Document Organizer

NLP + computer vision CLI tool that auto-classifies and files your documents

A command-line tool that intelligently organizes files in a directory using zero-shot NLP and image scene recognition — no manual labeling or predefined rules required. It reads the semantic content of documents and the visual context of images to determine appropriate categories, then moves files automatically. A persistent SQLite database tracks scan history to avoid re-processing, and a scheduler enables fully hands-free, recurring folder management.

Key Contributions
  • Implemented zero-shot text classification using facebook/bart-large-mnli via HuggingFace Transformers — no training data required.
  • Integrated OpenCV-based image scene recognition for classifying photos and visual files by content.
  • Built perceptual hashing for duplicate image detection, preventing redundant file copies.
  • Designed a SQLite persistence layer for scan history, enabling incremental runs without re-processing old files.
  • Added dry-run and watch modes via the schedule library for safe previewing and automated recurring execution.
